# paymentscan.bot# Payment Bot 💳

A simple, modern payment selection and processing interface built with HTML, CSS, and JavaScript.

## Features

- 🎨 Modern and responsive UI
- 📱 Mobile-friendly design
- 🔐 QR code payment option
- 📦 Multiple package selection (Basic, Standard, Premium)
- ✅ Payment processing simulation
- 🎯 Easy-to-use interface
